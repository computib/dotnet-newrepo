## About The Project


### Installation

```bash
dotnet tool install dotnet-newrepo -g
```

<!-- USAGE EXAMPLES -->
## Usage

You will need to have the "dotnet" and "git" commands in the path

```bash
mkdir Organization.Project
cd Organization.Project
dotnet newrepo
```

or more simply

```bash
dotnet newrepo Organization.Project
```
